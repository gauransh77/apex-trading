## 🚀 Deploy to GitHub Pages (5 minutes, free forever)

### Step 1 — Create a GitHub Account
Go to **github.com** → Sign up (free). Remember your username.

### Step 2 — Create a New Repository
1. Click the **+** button (top right) → **New repository**
2. Repository name: `apex-trading` (or any name you like)
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload the File
1. In your new repository, click **Add file** → **Upload files**
2. Drag and drop `index.html` from your Downloads
3. Scroll down, click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repository **Settings** tab
2. Scroll to **Pages** section (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Branch: **main**, Folder: **/ (root)**
5. Click **Save**

### Step 5 — Get Your URL
Wait 1-2 minutes, then your platform is live at:
```
https://YOUR-USERNAME.github.io/apex-trading/
```
**Bookmark this URL** — it's your permanent trading platform!

## 🔑 Setup After Deployment

### Live Prices (Twelve Data)
1. Sign up free at **twelvedata.com**
2. Go to Dashboard → copy your API key
3. In the platform, click **API KEY** → paste key → **Connect Live**
4. Prices update every 60 seconds on weekdays automatically

### AI Features (Anthropic)
When you first open the platform on GitHub Pages, a setup banner appears:
1. Go to **console.anthropic.com** → Sign up free (includes free credits)
2. Go to **API Keys** → **Create Key**
3. Paste the key in the setup banner → **Save & Activate**
4. All AI features now work: Generate Setups, Ask Marcus, AI Scanner

Both keys are saved in your browser's local storage — you only enter them once.
